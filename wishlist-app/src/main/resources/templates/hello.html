Wishlist

Автор: Дарья Баханкова tg: @dashbah

Реализованные функции:

  • Аутентификация и регистрация пользователей и контроль доступа к API
  • Поддержка gRPC или HTTP API (минимум 4 бизнес-метода)
  • Использование внешней БД для хранения данных
  • Схема базы данных создаётся при запуске или деплое приложения, поддерживается версионирование схемы
  • Автоматическое создание и версионирование схемы БД
  • Логирование - Loki
  • Метрики - Prometheus, OpenTelemetry Collector
  • Запуск в Kubernetes
  • CI интеграция с автоматической сборкой после попадания коммита в мастер
  • Swagger-документация для всех API-методов

Дополнительно реализовано:

  • хтмл страничка :)
  • Деплой в Yandex Cloud
  • Настройка ingress доступа к сервисам
  • Настройка nginx для внешнего доступа к машине
  • OpenTelemetry collector - сборка трейсинга и метрик (Zipkin, Jaeger) - внутри docker-compose
